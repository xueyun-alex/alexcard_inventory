# 产品类搜索与重置

## 界面与交互
- 修改 [ui/product_tab.py](ui/product_tab.py)，将左侧分类区包装为独立面板，在分类列表正上方加入 `QLineEdit`、`搜索`、`重置`；支持点击“搜索”和在输入框按 Enter 执行。
- 搜索词去除首尾空格并采用不区分大小写的包含匹配。父类命中时展示父类及其全部子类；子类命中时保留所属父类作为层级上下文，并仅展示命中的子类。
- 搜索期间只展示匹配的产品类分支；若当前选中项仍在结果中则保持选择，否则选中首个结果并刷新右侧产品。无结果时列表为空，右侧产品清空。
- 点击“重置”清空输入框、取消搜索条件、恢复“全部 / 未归类 / 完整父子类列表”，并恢复默认选择与产品显示。

## 刷新兼容
- 重构 `refresh_categories()` 的列表填充逻辑，让新增、重命名、删除、库存变化及手动刷新时继续应用当前搜索词，同时保留现有父类汇总、子类自身统计、缩进显示和筛选数据。
- 搜索仅在 UI 层过滤已有 `models.list_categories()` 结果，不修改数据库结构或查询接口。

## 验证
- 运行 Python 编译检查和现有可用检查。
- 手动验证父类命中、子类命中、部分匹配、空白词、无结果、Enter 搜索，以及重置后输入框和分类/产品筛选完全恢复。
