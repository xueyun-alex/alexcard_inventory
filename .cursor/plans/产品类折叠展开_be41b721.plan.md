# 产品类折叠与展开

## 实现
- 修改 [ui/product_tab.py](ui/product_tab.py) 的 `ProductTab`，增加会话级父类展开集合，初始为空，因此首次加载只显示“全部”“未归类”和父类。
- 在 `_build_ui()` 为产品类列表绑定点击事件；点击父类时切换其展开状态并重新渲染列表，点击子类及特殊筛选项仍维持现有行为。父类名称增加展开/收起标识，子类继续使用现有缩进样式。
- 调整 `refresh_categories()`：始终渲染父类，仅渲染已展开父类下的子类，并在刷新后保持当前父类或可见子类的选择；收起当前子类所属父类时改为选中该父类，避免筛选意外跳到“全部”。
- 兼容搜索：搜索命中的父类分支或子类所在分支在搜索结果中自动显示；重置搜索后回到用户原有展开状态。新建子类成功后自动展开对应父类，便于看到新增结果。

## 验证
- 运行 Python 编译检查并查看 [ui/product_tab.py](ui/product_tab.py) 的静态诊断。
- 手动验证：初始全部收起；父类连续点击可展开/收起；父类和子类筛选结果正确；收起已选子类时切回父类；搜索父类/子类及重置正常；刷新、新建子类、右键菜单和拖放归类不受影响。
