# 产品类旁显示库存总数

## 实现
- 在 [db/models.py](db/models.py) 增加按 `category_id` 聚合的统计查询，一次取得每类的产品数量 `COUNT(*)` 与库存总数 `COALESCE(SUM(stock), 0)`；无产品的类别由界面按 0 展示。
- 在 [ui/product_tab.py](ui/product_tab.py) 的 `refresh_categories()` 使用聚合结果，将列表文案统一调整为 `名称 (产品数)【库存 库存总数】`，覆盖“全部”“未归类”和所有已创建产品类。
- 在 [ui/product_tab.py](ui/product_tab.py) 完成手动库存调整后刷新类别统计；在 [ui/main_window.py](ui/main_window.py) 将入库完成后的产品页刷新改为 `refresh_all()`，保证入库、手动增减及操作记录回退后库存总数都不会滞后。

## 验证
- 运行 Python 编译检查。
- 启动应用，核对各类别、未归类及全部的库存合计；分别执行单个/批量库存调整、入库和回退，确认列表数字立即更新且产品数量保持正确。
